------------------ 两套配色 ------------------

  fig-XX.png        — 深色套(陶土暖色 / 偏深 / 重笔触)
                      默认使用,适配 dh-stage 黑金底
                      被 DhFigurine 自动加载

  fig-XX-light.png  — 浅色套(米白卷轴 / 偏浅 / 细线)
                      备用,适配未来浅底页面
                      需在调用时显式传入 src 属性:
                      <DhFigurine code="FIG-01"
                                  src="/figurines/fig-01-light.png" />

------------------ 框架逻辑 ------------------

DhFigurine 组件(src/components/atmosphere.tsx):
  · 优先用传入的 src
  · 否则按 code 拼路径:/figurines/{code.toLowerCase()}.png
  · 图加载失败时回退到 CSS 占位(斜纹方块)
